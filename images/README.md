# Images Folder

This folder contains visualizations generated during Exploratory Data Analysis (EDA)
for the Student Placement Analysis project.

## Included Visualizations

- **placement_status_distribution.png**
  - Shows the count of placed vs not placed students.

- **degree_vs_placement.png**
  - Compares degree percentage distribution based on placement status.

These plots help in understanding trends and patterns related to student placements.
